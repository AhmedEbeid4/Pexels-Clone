# Pexels-Clone

## Overview

An Android app developed with Kotlin that allows users to browse and download high-quality pictures and videos. Powered by <a href="https://www.pexels.com"> Pexels API</a>, users can discover stunning visuals, search for specific images and videos, and enjoy a seamless user experience. plus Clean Architecture, MVVM, and utilizes various libraries and technologies to ensure optimal performance and efficiency.


## Features

    Browse: Access a vast collection of high-quality pictures and videos

    Search: Effortlessly search for images and videos using relevant keywords to find exactly what you're looking for.

    Download: Download your favorite wallpapers to use as backgrounds or share with others.

    User Interface: The app boasts a visually appealing and user-friendly interface for an enjoyable browsing experience.
    
    Clean Architecture

    MVVM (Model-View-ViewModel)

 ## Technologies Used
 
    View Binding
    Dagger Hilt (Dependency injection)
    Retrofit
    Room
    Paging
    Kotlin Coroutines
    LiveData
    Flows
    Glide
    Navigation
